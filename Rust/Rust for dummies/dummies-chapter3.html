Speaking Rust: The Basics

Book 1, Chapter 3 of Rust All-in-One For Dummies by Paul McFedries teaches the foundational building blocks of the language: immutability by default, mutable variables with mut, constants, scalar data types (integers, floats, booleans, chars), and compound data types (tuples and arrays).

1. Declaring Mutable Variables

By default, variables in Rust are immutable (cannot be reassigned). To allow a variable's value to change, you must explicitly use the mut keyword during declaration.

src/main.rs — Mutability
Type the statement to declare a mutable variable named count set to 5:
let mut count = 5;

2. Defining Constants

Constants are always immutable and must be annotated with an explicit type. By convention, constant names are written in ALL_CAPS with underscores.

src/main.rs — Constants
Type the statement declaring a constant MAX_POINTS of type u32 equal to 100000:
const MAX_POINTS: u32 = 100000;

3. Scalar Data Types

Rust is a statically typed language. While Rust can infer types, you can explicitly annotate them. Scalar types represent a single value (integers, floating-point numbers, booleans, and characters).

src/main.rs — Floating Point
Type a variable declaration for price as a 64-bit float (f64) initialized to 19.99:
let price: f64 = 19.99;

4. Working with Tuples

A tuple groups together multiple values with a variety of types into one compound type. Tuples have a fixed length once declared.

src/main.rs — Tuples
Type a tuple tup containing an i32 (500), f64 (6.4), and u8 (1):
let tup: (i32, f64, u8) = (500, 6.4, 1);

5. Declaring Arrays

Unlike tuples, every element of an array must have the same type. Arrays in Rust have a fixed length specified at compile time inside square brackets: [Type; Length].

src/main.rs — Arrays
Type an array declaration numbers of 5 elements of type i32 initialized to 1 through 5:
let numbers: [i32; 5] = [1, 2, 3, 4, 5];
