Getting Your Hands Rusty

Book 1, Chapter 2 of Rust All-in-One For Dummies (Wiley) by Paul McFedries focuses on environment setup, using rustup, mastering the cargo toolchain, and writing your very first Rust executable project.

1. Creating a Cargo Project

Cargo is Rust's official package manager and build system. To initialize a new binary Rust project named hello_world, practice typing the precise CLI command below.

Terminal — CLI Command
Type the exact command to create a new project named hello_world:
cargo new hello_world

2. Fast Compilation Checks

To check your code for compilation errors without generating an executable binary (which saves significant time during development), run the check command.

Terminal — Cargo Workflow
Type the command to validate your code without compiling a full binary:
cargo check

3. Compiling and Executing

Cargo allows you to compile source files and immediately run the resulting binary in a single unified step.

Terminal — Build & Run
Type the command to build and run your Rust program:
cargo run

4. Writing the Main Function

Every executable Rust program requires a entry point function called main. In Rust, functions are declared using the fn keyword.

src/main.rs — Entry Point
Type the empty main function signature and braces:
fn main() {}

5. Complete "Hello, world!" Code

In Rust, println! is a macro (denoted by the exclamation mark !), not a standard function. Practice typing the full default main.rs program.

src/main.rs — Complete Script
Type the full program block below:
fn main() {\n println!("Hello, world!");\n}
