Speaking Rust: The Basics

Book 1, Chapter 3 of Rust All-in-One For Dummies by Paul McFedries teaches the foundational building blocks of the language: immutability by default, mutable variables with mut, constants, shadowing, scalar data types (integers, floats, booleans, chars), and compound data types (tuples and arrays).

1. Declaring Variables

Variables in Rust are created using the let keyword. Rust uses type inference to determine variable types automatically based on assigned values.

src/main.rs — Variables
Type the statement to declare a variable named score set to 100:
let score = 100;

2. Understanding Immutability

By default, variable bindings in Rust are immutable. Once a value is bound to a name, you cannot change that value later unless explicit mutability is specified.

src/main.rs — Immutability
Type an immutable binding for username set to "Alice":
let username = "Alice";

3. Making Variables Mutable

To allow a variable's value to change over time, add the mut keyword directly after let during declaration.

src/main.rs — Mutability
Type a statement declaring a mutable variable count set to 5:
let mut count = 5;

4. Working with Constants

Constants use const instead of let, must always be explicitly typed, and cannot be made mutable with mut.

src/main.rs — Constants
Type the constant MAX_POINTS of type u32 set to 100000:
const MAX_POINTS: u32 = 100000;

5. Shadowing Variables

Shadowing allows declaring a new variable with the same name as a previous one using let. It allows transforming values or types without making the variable mutable.

src/main.rs — Shadowing
Type a statement that shadows x by re-declaring it as x + 1:
let x = x + 1;

6. Exploring Data Types

Rust requires every value to have a known data type at compile time. Explicit type annotations use a colon after the variable name: let x: i32 = 5;

src/main.rs — Explicit Typing
Type a statement declaring age with explicit type u8 set to 25:
let age: u8 = 25;

7. Using Integer Types

Integers are numbers without a fractional component. They come in signed (i8 to i128) and unsigned (u8 to u128) variants. The default integer literal is i32.

src/main.rs — Integers
Type a statement declaring a signed 64-bit integer big_num initialized to -1000:
let big_num: i64 = -1000;

8. Using Floating-Point Types

Floating-point numbers are numbers with decimal points. Rust has two primitive float types: f32 and f64 (default).

src/main.rs — Floats
Type a statement declaring price as a 64-bit float (f64) initialized to 19.99:
let price: f64 = 19.99;

9. Working with Booleans

A Boolean type in Rust has two possible values: true and false. Booleans are 1 byte in size and specified using bool.

src/main.rs — Booleans
Type a statement declaring is_active of type bool set to true:
let is_active: bool = true;

10. Working with the Character Type

Rust's char type represents a Unicode Scalar Value (4 bytes). Literals are specified using single quotes, unlike string literals which use double quotes.

src/main.rs — Char
Type a statement declaring letter as type char initialized to 'R':
let letter: char = 'R';

11. Grouping Values with Tuples

A tuple groups multiple values with a variety of types into one compound type. Tuples have a fixed length once declared.

src/main.rs — Tuples
Type a tuple tup containing an i32 (500), f64 (6.4), and u8 (1):
let tup: (i32, f64, u8) = (500, 6.4, 1);

12. Storing Collections with Arrays

Unlike tuples, every element of an array must have the same type. Arrays in Rust have a fixed length specified at compile time inside square brackets: [Type; Length].

src/main.rs — Arrays
Type an array declaration numbers of 5 elements of type i32 initialized to 1 through 5:
let numbers: [i32; 5] = [1, 2, 3, 4, 5];
