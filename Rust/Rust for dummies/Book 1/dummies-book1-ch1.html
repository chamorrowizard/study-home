Hello, Rust! Getting Started

Book 1, Chapter 1 of Rust All-in-One For Dummies introduces the Rust toolchain: installing Rust via rustup, checking versions, writing a classic "Hello, world!" program, understanding the main function and macros, compiling with rustc, and creating and managing projects using cargo.

1. Verifying Rust Installation

Rust is installed and managed via rustup. To verify that the Rust compiler is successfully installed on your system terminal, you check its version using the version flag.

Terminal — Version Check
Type the command to check your installed Rust compiler version:
rustc --version

2. Defining the Entry Point

Every executable Rust program must have a main function. It serves as the initial code that runs when your program is executed. Functions are defined using the fn keyword.

src/main.rs — Main Function
Type the opening line for the main entry point function:
fn main() {

3. Printing Text with println!

println! calls a Rust macro (indicated by the exclamation mark !). Macros allow meta-programming and syntax extension; calling a regular function would omit the exclamation mark.

src/main.rs — Macro Call
Type the statement to print "Hello, world!" to standard output:
println!("Hello, world!");

4. Compiling with rustc

For simple single-file scripts, you can pass the source file directly to the compiler rustc to create a standalone binary executable file.

Terminal — Compilation
Type the terminal command to compile main.rs using rustc:
rustc main.rs

5. Creating a Project with Cargo

cargo is Rust's build system and package manager. It creates standard directory structures, manages dependencies via Cargo.toml, and builds your application.

Terminal — Cargo Initialization
Type the cargo command to create a new project named hello_cargo:
cargo new hello_cargo

6. Building Projects with Cargo

To compile a Cargo project without running it immediately, run cargo build. This generates an executable file in target/debug/.

Terminal — Cargo Build
Type the terminal command to build your Cargo project:
cargo build

7. Compiling and Executing with Cargo

cargo run conveniently compiles your code (if any changes were detected) and immediately runs the resulting executable binary in a single command.

Terminal — Cargo Run
Type the command to build and execute your Cargo project at once:
cargo run

8. Fast Verification with Cargo Check

cargo check quickly parses and checks your code for errors without taking the time to produce an executable binary, making it much faster during active development loops.

Terminal — Fast Verification
Type the command to check your project code without producing a binary:
cargo check
